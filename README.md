# ALX Project 0x00 Setup

This project is a simple Next.js app scaffolded using the `create-next-app` CLI with TypeScript.

## Features

- TypeScript
- ESLint
- Tailwind CSS
- Import Alias (`@/*`)
- No `/src` folder
- No App Router (using Pages Router)
